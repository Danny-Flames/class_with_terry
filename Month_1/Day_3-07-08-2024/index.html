HTML CSS

There are three ways to style your HTML document which are:

We'll be using 'ul' html elements which means 'Unordered list'.
The 'li' elements inside means 'List item'

  • Inline styling
  • Internal styling
  • External styling
Inline styling is the type of styling whereby you pass in the 'style' attribute to the opening tag and write all of your styles there. This text section is styled by an inline styling

Internal styling is the type of styling whereby you either give a 'class' or an 'id' attribute to the element you wish to style.

This is done by writing the styles for the element inside the 'style' element in the 'head' element of the HTML document.

This is a div with an id of play_box.

Note that when working with internal stylings, you use a dot(.) for an element with a 'class' attribute and a hash(#) for an eleent with an id attribute.

External styling are styles done in an external css style sheet.
This div element containing all these texts will be styled using an external styling method.
This method also uses 'id' or 'class' to style the element
